# OniBot

<p align="center">
 
</p>

<p align="center">
  A dark-themed desktop chatbot for Twitch, YouTube Live, and Discord.<br/>
  Drop-in replacement for DeepBot — with full data import.
</p>

---

## Download

Head to the [**Releases**](https://github.com/firstoni-dev/OniBot-releases/releases/latest) page to download the latest version.

Download `OniBot.exe` and run it — no installation required.

---

## Features

- Connects to Twitch, YouTube Live, and Discord simultaneously
- Full DeepBot data import — commands, chatters, points carry over
- Points and watchtime tracking per platform
- Custom commands with variables, counters, and repeat timers
- Betting, raffle, and queue systems for stream interaction
- Twitch EventSub integration — subs, cheers, raids, redemptions, hype train
- Helix API — badges, emotes, stream info, moderation
- Dark themed UI with CustomTkinter
